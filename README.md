# Karate Tournament Management System 🥋

A MERN stack application for managing karate tournaments efficiently. Features include online registration, AI-powered fair match draws, live scoring, real-time brackets, player stats tracking, and secure payments. Dashboards for players, referees, coaches, and organizers, plus an AI chatbot for guidance.

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Styling: CSS / Bootstrap
